## 🤝 כיצד להצטרף?

נשמח לקבל תרומות מכל סטודנט שרוצה לשפר, להוסיף או לתקן!

### צעדים לתרומה

1. **Fork** — לחץ על כפתור ה-Fork בפינה הימנית העליונה של הריפו
2. **Clone** — שכפל את הריפו למחשב שלך:
   ```bash
   git clone https://github.com/YOUR-USERNAME/student-notes.git
   ```
3. **Branch** — צור ענף חדש עם שם תיאורי:
   ```bash
   git checkout -b add/summary-data-structures
   ```
4. **Commit** — בצע את השינויים ושמור עם הודעה ברורה:
   ```bash
   git commit -m "Add: סיכום מבני נתונים — רשימות ועצים"
   ```
5. **Push & Pull Request** — דחף את השינויים ופתח Pull Request מול הענף `main`

### הנחיות תרומה

- ✅ ודא שהתוכן **מדויק ומובן** לסטודנט ממוצע
- ✅ השתמש בפורמט Markdown **אחיד** לפי שאר הקבצים בפרויקט
- ✅ כלול **דוגמאות קוד** בכל מקום שרלוונטי
- ❌ אל תעלה חומר מוגן בזכויות יוצרים ללא אישור
